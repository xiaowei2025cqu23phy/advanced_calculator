# Advanced Calculator (高级科学计算器)

这是一个功能强大的 Android 科学计算器应用，支持多种计算模式、公式预览以及 2D/3D 图形绘制。

## 主要功能

- **科学计算**：支持基础运算、三角函数、对数、幂运算等，具备 LaTeX 风格的公式预览。
- **复数运算**：支持复数的加减乘除及极坐标转换。
- **矩阵运算**：支持矩阵行列式、转置、求逆等常用操作。
- **图形绘制**：
  - **2D 绘图**：支持函数图像绘制。
  - **3D 绘图**：支持三维空间函数图像预览。
- **历史记录**：自动保存计算过程，方便随时查阅。

## 技术栈

- **开发语言**：Kotlin & Java
- **构建系统**：Gradle (Kotlin DSL)
- **核心库**：
  - **KSP (Kotlin Symbol Processing)**：用于高效的处理符号。
  - **Room**：本地数据持久化历史记录。
  - **mXparser**：强大的数学表达式解析库。
  - **EJML**：高效的矩阵运算库。
  - **MPAndroidChart**：用于 2D 图像绘制。
  - **OpenGL ES**：用于 3D 图像渲染。

## UI 优化

- **现代化键盘**：具有 3D 阴影质感的圆角按键，支持触感反馈。
- **响应式布局**：针对不同屏幕尺寸优化的显示区域。
- **暗色模式支持**：深度适配的暗色主题 UI。
